Testimony

As a student of web development, learning HTML and CSS has been an exciting and rewarding experience. HTML, or Hypertext Markup Language, is the foundation of web pages. I learned how to create the structure of a website using elements like headings, paragraphs, lists, and images. Writing HTML code felt empowering, as I could see my ideas come to life in a browser.

Once I grasped the basics of HTML, I moved on to CSS, which is used to style web pages. With CSS, I learned how to change colors, fonts, and layouts, making my websites visually appealing. I enjoyed experimenting with different styles and discovered how to use properties like margin, padding, and border to refine my designs. Learning about responsive design was particularly exciting, as I found out how to create websites that look good on any device, whether it’s a phone, tablet, or desktop.

Through hands-on projects and assignments, I gained practical skills in organizing content and enhancing user experience. Each project challenged me to think creatively and problem-solve, whether it was fixing layout issues or improving the overall design. I also learned the importance of writing clean, organized code, which makes it easier to maintain and update my websites.

Overall, my experience with HTML and CSS has given me a strong foundation in web development. I am now more confident in my abilities and excited to continue learning more advanced topics, such as JavaScript and back-end development. Each step I take in this journey helps me grow as a developer, and I look forward to the challenges and opportunities that lie ahead.
